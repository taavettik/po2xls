# Usage

Packaged binaries can be downloaded from [releases](https://github.com/taavettik/po2xls/releases)

```bash
# Basic usage
$ ./poxls-linux <source> <target>

# Converting xls to po
$ ./poxls-linux messages_DE.xlsx messages_DE.po

# Converging po to xls
$ ./poxls-linux messages_FI.po messages_FI.xlsx
```
